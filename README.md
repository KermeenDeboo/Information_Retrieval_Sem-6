# Welcome to the Information Retrieval Sem 6 repository!
This repo contains a collection of practicals related to core Information Retrieval concepts under subject 'USCSP6032 Information Retrieval – Practical' of 'TY BSc CS (Mumbai University)'. 
It's designed to be clean, beginner-friendly, and easy to navigate—perfect for students, learners, and anyone who wants to revise DS fundamentals with working code.

# 🚀 What's Inside?
You'll find implementations of the most essential information retrieval such as:
Document Indexing and Retrieval
Retrieval Models
Text Categorization
Web Crawling and Indexing
Link Analysis and PageRank
Advanced Topics in Information Retrieval etc.
Each practical is placed inside its own folder/file to keep things simple and clear.

# 🎯 Goal of This Repository
This repository aims to:
Help understand IR concepts through code
Provide clean and easy-to-read implementations
Encourage hands-on practice and experimentation
